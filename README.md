** 코드 속 작업 내용 **
1. 플레이어 기본적인 움직임 동작

2. 적 기본적임 움직임 동작

3. 사운드
    - 공격음, BGM, 타격음, 풋사운드
4. HUD
    - 체력바, 탄약 상태 정보
5. 아이템
    - 탄약
6. 게임모드
    - 모든 몬스터 클리어 시  성공, 그 전에 사망 시 실패
    - 리스폰 시간 후 재도전 기능
