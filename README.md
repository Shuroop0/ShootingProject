** 코드 속 작업 내용 **
1. 플레이어 기본적인 움직임 동작
    - 공격, 사망, 회복, 달리기, 점프, 걷기, 탄약 수집
2. 적 기본적임 움직임 동작
    - 플레이어 탐지, 공격, 사망, 달리기, 일정 시간 제자리로 스폰위치로 복귀
3. 사운드
    - 공격음, BGM, 타격음, 풋사운드
4. HUD
    - 체력바, 탄약 상태 정보
5. 아이템
    - 탄약
6. 게임모드
    - 모든 몬스터 클리어 시  성공, 그 전에 사망 시 실패
    - 리스폰 시간 후 재도전 기능
